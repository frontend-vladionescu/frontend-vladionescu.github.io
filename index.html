Lawyers

This web aplication contains a list of public defenders (lawyers) stored into an array of objects and generates 8 lists dynamically. Each list stores some periods in which the list is active by manipulating the Date() object. You can login as a police unit to solicit a public defender from the list or as the SAJ (the service that manages the public defenders) and you can manipulate the list by deleting, adding, editing or suspending a lawyer from the list
